# Perl文法メモ

標準入力

```perl
my $input = <STDIN>;
```

配列

```perl
my @array = (1, 2);
my ($a, $b) = split(/,/, $input);
```

改行コードを削除

```perl
chomp($input);
```

文字列比較 & ループ抜け

```perl
if ($input eq "n") {
	last;
}
```

標準出力+改行コード

```perl
say $input;
```
