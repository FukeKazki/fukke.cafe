neovimの設定について

# 隠しファイルを表示したい

```lua:title=neo-tree
filesystem={
  filtered_items={
    visible: true,
  }

```

# quick fix
LSPの機能を使う
`space l a`

# 保存する
`space w`

# 閉じる
`space q`

# コードの差分を一部戻したい
`space g h`

# 関数の情報とかをみたい
`shift k`

# vimの機能
# ファイル内の単語を置換したい
`:%s/置換したい単語/置換したあとの単語`
