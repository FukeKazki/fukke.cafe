# 環境

- vscode

# 関数名・変数名がわかってるとき

```bash
git grep -i 変数名
```

```bash
tig ブランチ名
g -i 変数名
```

vimっぽい画面に入ったときは`/hensuu`で検索する。

`command + shift + f`で変数名・関数名を検索。

# ファイル名がわかってるとき

`command + p`でファイル名を検索。

# ファイル特定後

`command + f`でファイル内検索。

各関数に`console.group('関数名')`をつけてデバッグ。
