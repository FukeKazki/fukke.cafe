# なにができる
Virtual Machine(VM)です。

世間ではDockerDesktopの代わりとしてlimaで立てたUbuntuにDockerをインストールして使われてます。

# 言葉の定義
- インスタンス (VMで立てたマシンのこと)

# いろいろ
- `limactl start template://ubuntu`

- `limactl ls`

- `limactl shell ubuntu`

- `limactl factory-reset ubuntu`

- `limactl start ubuntu`

- `limactl stop ubuntu`

- `limactl delete ubuntu`
