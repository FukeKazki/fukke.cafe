先日Nuxt3が公開されたので触ってみたメモです。

### データ取得
`useFetch`等を使ってSSR時にデータ取得ができる。

### API
`/server/api`フォルダに任意のAPIを生やせる。

### composable
カスタムフック等を入れると自動でインポートされて便利。

### components
コンポーネントを入れる。こちらも自動でインポートされる。

### layouts
画面全体のレイアウトを適用できる。ページごとに設定もできる。

### pages
ファイルルーティングしてくれる。内部はvue-router。

### ミドルウェア
クライアントでのミドルウェアとサーバーのミドルウェアがある。難しい。

### 認証
`useCookie`を使ってクライアント・サーバー両方から触れる。

### テスト
vitest等を使ってテストできるがまだBeta版。

### 画像最適化
nuxt-imageがあるがまだBeta版。
