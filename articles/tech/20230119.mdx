# Pick
Pickの実装です。`K extends keyof T`でKをTのプロパティにしています。
例えばTが`{a: string; b: string}`のときKは`a | b`になります。
`[P in K]`でKのUnionの数だけfor文みたいになります。`T[P]`で実際にアクセスしています。

```typescript
type MyPick<T, K extends keyof T> = {
  [P in K]: T[P]
}
```
