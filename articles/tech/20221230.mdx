## 今年一年を振り返って
気付けば12月30日の21時35分です。2023年になる前に今年学んだ技術を振り返ります。
今年は高専を卒業し、新社会人として働き始めた年です。
個人開発でこの技術触った！というのよりも業務を通して初めて触った技術がたくさんある印象です。
では振り返っていきましょう。

- playwright
  - e2eといえば！ビジュアルリグレッションテストでおせわになりました
- Google Search Console
  - 毎日にらめっこしてました
- CoreWebVitals
  - CLS, LCP等々ずっと監視・改善してました
- AWS Amplify
  - AWSは詳しくないですが静的サイトのデプロイはできるようになりました
- lima-vm
  - M1Macでも使えるVMです、便利
- Unity
  - Macの性能が上がってきたのでUnityで遊び始めました
- Angular
  - オワコンと呼ばれているAngularですがクリーンアーキテクチャを学んでからは楽しくて好きです
- RxJS
  - 初めてのリアクティブプログラミング。たのしいです
- GitLabRunner
  - CircleCI記法でRunnerを書いたりしてました
- SPA
  - SPA独特の知識が増えました。まだまだ捨てたものじゃないです
- Figma
  - UIデザイナーの真似事してました、Angularコンポーネントを生成するプラグインも作りました
- Docker
  - イメージを作ってアップロードして等いままで雰囲気で使っていたところのレベルが上がりました
- TestCafe
  - E2Eテストツールです。クセはあります
- Jest
  - 業務になってからUnitテストを書くようになりました
- Gulp
  - 昔ながらのタスクランナーツール
- Nuxt3
  - Nuxt3がきてからがVue時代の幕開け！使いやすかったです
- Yarn3
  - いろいろなプロダクトにYarn3を導入しました。中にはゼロインストールしてみたものもあります
- Flutter
  - Androidを持つようになってFlutterを触りました。でもアプリ開発は難しい
- WebAssembly
  - WebAssemblyの時代がくる！って言われたので学びました。JSから読み込んで実行までですが
- Azure Custom Vision
  - 実家のねこ3匹判定BOTを作るときに使いました。写真をアップロードしてラベル付けするだけで使えるので便利！
- Firebase
  - 昨年に続けてとてもお世話になりました。プロジェクト作りすぎて上限になったのが今年の思い出です。
- OCR
  - 文字起こしばりぐっとくん勉強会後、一時期OCRにハマってました。レシート読み取りBOTを作りました
- セグウェイ
  - オフィスが広くなってセグウェイ乗り放題になったので完全に習得できました。セグウェイ4足歩行できます！
- Oculus Quest2
  - AFrameでマインクラフトを作ったりUnityで遊んだりしました
- zig
  - WebAssemblyの生成にZig言語を勉強しました
- RaspberryPi
  - 日本語対応電光掲示板の作成で触りました
- 配信周り
  - hlsやRTMP, gRPCについて勉強しました
