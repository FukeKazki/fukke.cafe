neovimの設定について

基本はastronvimを使っている

# 隠しファイルを表示したい

```lua:title=neo-tree
filesystem={
  filtered_items={
    visible: true,
  }
}
```

# ファイル分割
`shift | or \`

# quick fix
LSPの機能を使う
`space l a`

# 保存する
`space w`

# 閉じる
`space q`

# コードの差分を一部戻したい
`space g h`

# 関数の情報とかをみたい
`shift k`

# vimの機能
# ファイル内の単語を置換したい
`:%s/置換したい単語/置換したあとの単語`

# インデントを深くする
`>G`

# 直前の操作を繰り返す
`.`

# 現在の行の末尾にinsert
`$a`のマクロ
`A`

# 現在の行で検索する
`f単語`
次の単語へ移動: `;`
前の単語へ移動: `,`

# カーソル位置の文字を削除してinsertモードにする
`s`

# カーソル位置の単語を削除してinsertモードにする
`cw`

# カーソル位置の単語を検索
`*`
次の単語へ移動: `n`

# カーソル位置の単語を削除する
`daw` delete a word
