# よかった
メガネを掛けるだけのAR体験は初めてのものだった。
Oculus等に比べて軽くてつけ心地がいい。
カバーを付けてリアルを遮断すると没入感がUPする。
80度ぐらいしか覆わないので外の景色は見える。VR、ARゴーグルほどの没入感はない。

# まだまだ
そもそも眼球での視点移動になるのでめっちゃ疲れる。
有線接続なのでちょっと煩わしい。
モニターにしたときに4Kサイズじゃないからちっちゃい。
